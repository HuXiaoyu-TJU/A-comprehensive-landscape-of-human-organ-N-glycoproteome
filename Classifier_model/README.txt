###Summary###
The function of this code is to generate classifier model.


###Depedencies###
Python is required.
Package "scikit-learn" (v1.3.0) was used for machine learning pipelines, feature selection, and model evaluation. "pandas" and "numpy" were utilized for data manipulation and numerical computations. "matplotlib" was employed for data visualization and ROC curve generation.

###Protocol Steps###
# It does not need to be installed on the desktop, and can be used once python is configured.
# Select all codes and click "run", these results are generated automatically.

# After running the program, you will get: 
1) ROC Data of all models.
2) ROC Plot of all models.
3) ROC Plot of all organs of each model.
4) model_evaluation_results.
